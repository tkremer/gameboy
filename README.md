This is the software for an arduino-base simple game console.
It includes Snake, Tetris and the Game of Life, as well as a basic oscilloscope
program to be used with the analog inputs of the arduino.

Recently, additional applications to test new hardware have been added:
- Servo: control a servo.
- 1-Wire: Read 1-wire device roms and DS80B20 temperature sensors.
- Pinpad: Read from the pinpad from my fablock project.
